# 🏦 Banking System (C++ OOP Project)

## 📌 Description

This is a menu-driven banking system built using C++ concepts.
It demonstrates Object-Oriented Programming including inheritance, polymorphism, and file handling.

---

## 🚀 Features

* Create Account
* Deposit Money
* Withdraw Money
* Display Account Details
* Calculate Interest
* Transfer Money
* Save Data to File
* Load Data from File

---

## 🧠 Concepts Used

* Classes and Objects
* Inheritance
* Virtual Functions (Polymorphism)
* File Handling (fstream)
* Vectors (STL)

---

## ⚙️ How to Run

1. Compile:

```bash
g++ main.cpp -o bank
```

2. Run:

```bash
./bank
```
